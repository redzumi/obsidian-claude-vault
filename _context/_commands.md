# Commands

Available Claude Code slash commands for this vault.

| Command | Description |
|---------|-------------|
| `/morning` | Morning briefing: Calendar + Todoist + focus + stale project detection |
| `/dayend` | End-of-day closing: capture what was done, update session-context |
| `/weekly-review` | Weekly sync: Todoist Waiting For + project check-in + next week's focus |

## How to run

In Claude Code terminal:
```
/morning
/dayend
/weekly-review
```

Or trigger via natural language: "let's do morning review", "close the day", "weekly review".

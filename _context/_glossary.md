# Glossary

Mini-dictionary so Claude doesn't have to guess meanings.

| Term | Meaning |
|------|---------|
| `vault` | This Obsidian repository with PARA structure and Claude automation |
| `PARA` | Projects / Areas / Resources / Archive — how to organize notes |
| `GTD` | Getting Things Done — task management methodology (used in Todoist) |
| `session context` | 5-10 lines: project status, next step, links to sources |
| `weekly review` | Ritual: Todoist sync + project check-in + next week's focus |
| `focus` | Todoist label for tasks that are part of the weekly focus |
| `L0 / L1 / L2` | Token-lean loading levels — how much context Claude reads per session |
| `vibe coding` | Development style with heavy AI assistance |

Add new terms here as your system evolves.
